Users Table
Path Method Description
/users GET Gets all the users
/users/:id GET Gets a single user by id
/insert POST Adds a user to the database
/delete/:id DELETE Deletes a user by ID
/edit/:id PATCH Edits all information of a user by id
Products Table
/products GET Gets all the products in the database
/products/:id GET Gets a single product by id
/insert POST Adds a product
/delete/:id DELETE Deletes a product by ID
/edit/:id PATCH Edits information of a product by id
